SF Film Finder
==============
App to find locations of movies filmed in San Francisco. (Experiment to try some technologies)
Data parsed from [San Francisco Data](https://data.sfgov.org/)

Backend
-------
* Python
* Flask
* PostgreSQL

Frontend
--------
* Backbone
* Foundation
